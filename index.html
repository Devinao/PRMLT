Welcome to PRMLT

This a Matlab package which implemented the algorithms described in the classical machine learning textbook PRML (Pattern Recognition and Machine Learning). The package is self-contained and purely Matlab based, there is no outside package dependency and no mex-based native (like C++) code inside.

The code is heavily commented and the symbols are synced with the book, in order to make code easy to read and learn.

Installation and Usage

(1) Download the zip file above and extract it wherever you want.

(2) Navigate Matlab to package location as working directory and run the init.m script.

(3) Play with any function in the package.
